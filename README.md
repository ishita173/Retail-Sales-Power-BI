# Retail Sales Analysis Dashboard (Power BI)

## 📌 Objective
Analyze customer purchasing behavior and identify high-value customers and revenue trends.

## 🛠 Tools Used
- Power BI
- DAX
- Data Cleaning

## 📊 Key Insights
- Top 10 customers contribute a large share of total revenue
- UK has highest number of customers
- Ireland shows highest average revenue per customer
- Seasonal demand peaks during specific months
  
## 🚀 Features
- Interactive filters (country, time, product)
- KPI tracking (Revenue, Customers)
- Customer segmentation
